/*
 enum(열거형) => 서로 관련있는 상수들의 집합을 나타낸다
              => 클래스처럼 보이게하는 상수
              => 
                 - 클래스처럼 독립된 java파일에 만들 수 있다 
                 - 하나의 java파일에 클래스와 같이 만들 수 있다
                 - 클래스 안에 내부 클래스처럼 만들 수 있다
                 
 - 열거형의 속성 및 메소드
   - name()                             => 열거형 상수의 이름을 문자열로 반환
   - ordinal()                          => 열거형 상수가 정의된 순서값(index)을 반환 (0부터 시작)
   - 열거형이름.valueOf("열거형상수명") => 지정된 열거형에서 '열거형상수명'과 일치하는 상수를 반환
   - 열거형이름.상수명                  => valueOf와 동일
   - 열거형이름.values()                => 모든 열거형 상수들을 배열로 반환
 
 - 열거형 선언
    1)
      enum 열거형이름 {상수명1, 상수명2, ...}
    2)
      enum 열거형이름 {
      	상수명1(값1, 값2, ...),
      	상수명2(값1, 값2, ...),
      	...
      	상수명n(값1, 값2, ...);
      	
      	//값들이 저장될 변수
      	private 자료형이름 변수명;
      	...
      	
      	//열거형의 생성자 ('열거형 상수'의 '값'들을 변수에 세팅하는 역할)
      	//열거형 생성자는 묵시적으로 private 이다
      	//'변수명'의 개수는 '값'들의 개수와 같고 자료형도 각각 일치해야 한다
      	private 열거형이름(자료형 변수명, ...) {
      		위에 선언된 변수들을 초기화
      	}
      	
      	//구성된 '값'들을 외부에서 불러올 수 있는 getter를 만든다
      } 
 */
